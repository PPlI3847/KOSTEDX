# 1. 문자 생성 방법

콧테드를 표기 가능한 문자는 라틴어뿐만 아니라 콧테드자도 있다. 유니코드에 이를 직접 등재를 하려면 복잡한 절차를 거쳐야하기 때문에 라틴어 기본 폰트인 Arial을 바탕으로 대문자 A~Z, 소문자 a~z만 변경을 하여 완성했다. 참고로 KOSTEDX 폰트를 제작할 때 시행착오를 굉장히 많이 겪었다. 다음은 제작 과정 순서이다.

1. 우선 영어 알파벳에 대응되는 새로운 문자들을 아이패드 굿노트에 그린다.
2. 굿노트에서 PDF로 저장을 누르고 컴퓨터[^1]로 이동한다.
3. Fontforge와 Inkscape를 다운받은 뒤 Inkscape로 아까 그린 문자 계획서 pdf를 불러와 밑에 두고 벡터로 따라 그린다.[^2]
4. 컴퓨터에 내장되어 있는 Arial을 복사한뒤 Fontforge로 들어와 복사된 Arial를 연다.
5. 만들어 놓은 벡터기반 문자를 각각 대문자 A~Z, 소문자 a~z에 알맞게 넣는다.[^3]
6. 필요한 문자를 모두 넣었다면 이제 Fontforge 홈으로 돌아와서 Metrics의 Set Both Bearing을 조절해 각 문자 사이의 간격을 같게 한다.
7. Element의 Font Info에 들어가 폰트 정보를 입력한다.
8. File의 Generate Fonts에 들어가 Options 위를 눌러 파일 저장 방식을 TrueType을 변경하고 폰트 이름을 정한 뒤 Generate를 누른다.
9. 에러 관련 메세지가 떠도 그리 심각하지는 않으니 무시한다.
10. 각 운영체제에 맞는 폰트 적용 방식을 사용한뒤 로컬에서 사용가능한 문서 편집기를 사용해 폰트가 잘 작동하는지 확인한다.

자음은 b, d, g, h, j, k, m, n, p, r, s, t 총 13개이다.

받침의 역할은 이들 중 b, g, k, m, n, r, s 만 가능하며 다른 알파벳은 받침의 역할을 하지 못한다.

자음 뒤에 자음이 오는 형식에서 앞의 자음을 받침으로 취급한다 이외의 상황은 연음한다 접미사는 연음을 하지 않는다.

* ab = 압
* ag = 앙
* ak = 악
* am = 암
* an = 안
* ar = 알
* as = 앗

모음은 a, e, i, o, u, w, x, y 총 8개이다.

w와 y는 복모음으로만 사용된다.

* wa = 와, we = 외, wi = 위, wo = 워 
* ya = 야, ye = 예, yo = 요, yu = 유
